Is there an official mempool app for iPhone?

Last updated June 2026 · 4 min read

Short answer: there's no official mempool.space app for iOS. mempool.space ships an official Android app, but iPhone users are sent to the website, unless they use a native third-party app like Bitcoin Insight.

In a nutshell: mempool.space → website + Android app, no iOS app. For a native iPhone experience, Bitcoin Insight brings the same live mempool, fees and block-explorer data to iOS, free, open source, no subscription.

The short answer

If you've searched the App Store for a "mempool" app, you've probably noticed something: there is no official mempool.space app for iPhone. The mempool.space team maintains the web explorer everyone knows and an official Android app, but iOS has been left to the browser.

That gap is exactly why native iOS apps like Bitcoin Insight exist: to give iPhone users a fast, glanceable, app-grade way to follow the mempool, network fees and the rest of the Bitcoin network, using the same public mempool.space data underneath.

Does mempool.space have an app?

Partly. Here's the current state of play:

  • Website, mempool.space is the canonical Bitcoin explorer, available in any browser and installable as a PWA.
  • Android, there's an official "Bitcoin Explorer by mempool" app on Google Play.
  • iOS / iPhone, there is no official native app. You can add the website to your Home Screen, but it isn't a real iOS app with widgets, haptics and native navigation.

Why there's no official iOS app

mempool.space is an open-source project focused on the explorer and its backend infrastructure. A native iOS app is a separate, ongoing commitment, App Store review, WidgetKit, StoreKit, biometric APIs, so for now iPhone remains web-only on the official side. The good news: because mempool.space exposes a clean, public, no-account API, independent apps can build a great native iOS experience on top of it.

FAQ

Is there an official mempool.space app for iOS?

No. mempool.space publishes an official Android app, but there is no official native mempool app for iOS. On iPhone you either use the mempool.space website or a third-party native app such as Bitcoin Insight, which is built on the public mempool.space API.

What is the best mempool app for iPhone?

Bitcoin Insight is a native iOS app that brings live mempool data, network fees, a block explorer and watch-only wallets to iPhone. It is free, open source, has no ads, no accounts and no subscription, and it uses mempool.space as its data source.

Is Bitcoin Insight affiliated with mempool.space?

No. Bitcoin Insight is an independent, open-source app that uses the public mempool.space API for data. It is not affiliated with or endorsed by mempool.space or Apple.
